# Twitch Alert Overlays

Este repositório contém uma coleção de overlays customizados para transmissões na Twitch. O objetivo é centralizar diversos elementos visuais e alertas que podem ser integrados diretamente em softwares de transmissão.

## Estrutura do Projeto

O projeto é organizado em diretórios, onde cada pasta na raiz representa um componente de overlay específico (ex: alertas de seguidores, inscrições, doações, etc).

Cada diretório de overlay geralmente contém:
- `index.html`: O arquivo principal que deve ser carregado no software de transmissão.
- `style.css`: Responsável pela aparência e pelas animações do elemento.
- `assets/`: Recursos de mídia como áudios, imagens ou fontes utilizados por aquele overlay.

---
*Este repositório está em constante evolução, com novos overlays sendo adicionados periodicamente.*
